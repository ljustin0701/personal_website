Blog

Something amazing I found that I would like to share with you is not really a technology but a software library that is used in Android technology. I came across a very useful library called Volley. Volley was created by Google to make http requests and other networking needs easier and faster to do. Before Volley, http requests were usually handled by creating an AsyncTask which allows you to perform background operations and lets you show the results in the UI. When using AsyncTask, a lot of boiler-plate code is needed to be written in order to do the operation you intend to do. Volley handles the boiler-plate code for you and lets you focus on the task itself such as making http requests. Some of the benefits of using Volley over AsyncTask includes :

  • Automatic scheduling of network requests
  • Being able to handle multiple concurrent network requests
  • The ability to cancel requests if you desire to
  • Increased customization for cases of retry or backoff for network requests
